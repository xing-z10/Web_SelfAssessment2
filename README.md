# Web Javascript and DOM Self Assessment

## AirBNB San Francisco – First 50 Listings

A tiny single-page app that loads the first 50 entries from `airbnb_sf_listings_500.json` via `fetch`/`await` and renders them as Bootstrap cards.

### Features
- AJAX loading of the first 50 listings
- Responsive grid (3 columns on desktop → 1 on mobile)
- Creative hover lift effect
- Direct link to each listing on Airbnb
